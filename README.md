# 🚀 AI Search Audit

Get your LLM Visibility Report in seconds.

AI Search Audit analyzes websites from the perspective of AI-powered search systems such as ChatGPT, Gemini, Claude, Perplexity, and Google AI Search.

Unlike traditional SEO tools, AI Search Audit focuses on how well a website can be understood, trusted, cited, and used by modern AI search engines.

## Features

### AI Visibility

* AI Visibility Score
* Entity Clarity Audit
* Citation Readiness Audit
* Answer Extraction Audit
* Trust Signals Audit

### SEO & Technical Analysis

* SEO Health Audit
* robots.txt Analysis
* sitemap.xml Analysis
* Open Graph Audit
* Twitter Card Audit
* Schema Markup Audit
* Advanced Schema Audit
* Content Structure Audit
* WCAG 2.2 Audit

### Reporting

* LLM Visibility Report
* Dynamic Executive Summary
* Priority Issue Ranking
* Actionable Recommendations
* Category Detail Reports

### Sitewide Intelligence

* Multi-page Website Crawling
* About / Contact / Services Discovery
* Sitewide Entity Analysis
* Sitewide Trust Analysis
* Sitewide Citation Analysis
